# Mos3ef Project

## Overview
**Mos3ef** is a full-stack web application designed to connect patients with hospital services.

## Project Structure

- **`/backend`**: .NET 8 Web API Solution.
  - `src/`: Contains the source code for API, BLL, and DAL.
- **`/frontend`**: React 19 (Vite) Application.
- **`/docs`**: Project documentation. [View Full Documentation](docs/Project_Documentation.md)

## Getting Started

### Backend
Navigate to `backend/src/Mos3ef` and run:
```sh
dotnet run
```

### Frontend
Navigate to `frontend` and run:
```sh
npm install
npm run dev
```
